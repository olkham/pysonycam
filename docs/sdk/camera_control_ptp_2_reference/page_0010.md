DigitalImagingDesc.xml
The DigitalImagingDesc.xml file contains details about the service. This section explains the key tags
necessary for PTP-IP connection. If a tag does not exist, the function is not supported.
X_DigitalImagingDeviceInfo: Contains device information.
X_ModelName: Model name of the device.
X_ServerVersion: Server version; versions 3.00 and above support PTP-IP.
X_SerialVersion: The camera's serial number.
X_PTP_Information: Provides PTP-related information.
X_PTP_Versions: Supported PTP version.
