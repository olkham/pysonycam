To capture the picture, start recording the movie, lock AE or AF, and the Initiator sends Controls to the

The protocol is described in detail in the following section.
For Connections Using PTP-IP
PTP-IP is available when connected via Wi-Fi Direct (AP Mode), Wi-Fi ST Mode, or Wired LAN. The
designated port number for PTP-IP is TCP 15740 (for SSH connections, please refer to Connection by SSH).
For detailed information regarding the connection interface, please refer to the help guide for each model.
When connecting via PTP-IP, the following settings must be set on the camera: ILCE-1 is used as an example
here. For other models, refer to the "PC Remote Function" in the Help Guide.
Please set "Network > Cnct./Remote Sht. > Remote Shoot Function > Remote Shooting" to "On".
