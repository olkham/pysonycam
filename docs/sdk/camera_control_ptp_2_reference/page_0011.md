X_PTP_PairingNecessity: Pairing operation information; if 'Necessary', pairing operation is
required.
X_PTP_MediaServerSupport: If 'Enable,' supports Contents Transfer Mode.
X_PTP_RemoteControlSupport: If 'Enable,' supports Remote Control Mode.
X_ConnectionInfo: Contains connection information.
X_SSH_Support: SSH connection information; if 'Enable,' SSH connection is required.
