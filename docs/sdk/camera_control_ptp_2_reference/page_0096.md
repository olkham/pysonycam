SDIE_ObjectRemoved
Summary
Notify that a shot file is deleted.
The device sends this event when the shot file is transferred.
Description
```c
Event Code: 0xC202
```
Parameter1: ObjectHandle
Parameter2: None
Parameter3: None
Note
The host must not send GetObjectInfo with the notified ObjectHandle after receiving this event.
