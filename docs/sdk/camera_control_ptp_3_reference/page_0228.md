Image ID (String)
Note
This device property is used only when the device property Image ID (Numerical Value) Setting is on
```c
(0x02).
```
For the maximum character size that can be set, see Maximum Size of Image ID String. Please set the
string in UTF16BE.
Refer to Linking GPS Information with Shooting Images in Tips for how to use it.
By specifying a value in this property before shooting, the value specified in the Exif tag of the image file
shot after that will be saved. If you shoot immediately after setting, it may not be recorded on the Exif tag.
Be sure to obtain and ensure that the set and get values match before shooting.
```c
The Exif tag for the Image ID (String) is 0x2043.
```
