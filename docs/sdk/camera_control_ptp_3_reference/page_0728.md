Linking GPS Information with Shooting Images
After shooting, if the information recorded in the Exif tag of the image file is collated with the GPS
information, the image can be synchronized with the GPS information with high accuracy by using the
```c
“ImageID” feature. The ImageID provided is stored in the Exif data of the image, in Exif tags 0x2042 and
0x2043. After the flight, post-processing is required to reconcile the GPS data stored in the host with each
```
image, using ImageID as a reconciliation index.
