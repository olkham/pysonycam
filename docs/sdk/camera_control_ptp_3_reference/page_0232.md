Value
Description
```c
0x00000000 ~ 0xFFFFFFFF
```
The cache size [Upper 8bits], Cache rec time(1/100sec) [Lower 24bits]
Note
The cache size is set in the upper 8bits, and the cache rec time is set in the lower 24bits.
```c
If the cache size is “Short” and cache rec time is 5sec, set 0x010001F4. 0x01 = “Short,” 0x0001F4 =
```
500/100 = 5.00sec
```c
If the cache size is unused and only the cache rec time is set, set 0x00 in the upper 8bits.
The value of cache size is “None(0x00),” “Short (0x01),” “Middle (0x02),” “Long (0x03),” “Max (0x04).”
0x05 to 0xFF is the reserve value.
```
Ex.) If the camera supports “Short - 5sec,” “Middle - 10sec,” “Long - 20sec,” and “Max - 31sec” as cache
size and rec time, Enum values are the following.
```c
Enum value[0] = 0x010001F4 (Short - 5sec)
Enum value[1] = 0x020003E8 (Middle - 10sec)
Enum value[2] = 0x030007D0 (Long - 20sec)
Enum value[3] = 0x04000C1C (Max - 31sec)
```
