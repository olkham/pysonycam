Note
The AF Free Size and Position Setting (height, width) [Upper 4 bytes]: the height is set in the upper two
bytes, and the width is set in the lower two bytes.
The AF Free Size and Position Setting (x, y) [Lower 4bytes]: the x coordinate is set in the upper two bytes,
and the y coordinate is set in the lower two bytes.
Separated min/max/step values for each “height,” “width,” “x,” “y” for AF Free Size and Position range.
Set the range value for each.
For the set of Free Size 1, Free Size 2, and Free Size 3, refer to Focus Area. When each Free Size Setting is
selected, the selected Free Size Setting can be changed.
