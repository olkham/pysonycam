Device Discovery
Device discovery uses UPnP. Receive the service name USN urn:schemas-sony-
com:service:DigitalImaging and retrieve the dd.xml (Device Description file).
Example SSDP header:

## NOTIFY * HTTP/1.1


## HOST: 239.255.255.250:1900

CACHE-CONTROL: max-age=1800
LOCATION: http://192.168.122.1:80/dd.xml
NT: urn:schemas-sony-com:service:DigitalImaging:1
NTS: ssdp:alive
SERVER: UPnP/1.0 SonyImagingDevice/1.0
```c
USN: uuid:00000000-0000-0000-0000-000000000000::urn:schemas-sony-com:service:DigitalImaging:1
```
From dd.xml, confirm friendlyName and UDN, and obtain the path to DigitalImagingDesc.xml from the
SCPDURL tag.
