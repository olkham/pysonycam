| 0x0000000000000000 | Invalid |
| Other than above values | The real value of shutter speed (Upper four bytes: numerator, Lower four bytes: denominator) In the case of the shutter speed is displayed as “Real Number” on the camera, the denominator is fixed 0x0000000A. e.g.) 0x0000000F0000000A: 0x0000000F (means 15) / 0x00000000A (means 10) = 1.5" In the case of the shutter speed is displayed as "Fraction Number" on the camera, the numerator is fixed 0x00000001. e.g.) 0x00000001000003E8: 0x00000001 (means 1) / 0x000003E8 (means 1000) = 1/1000 |
