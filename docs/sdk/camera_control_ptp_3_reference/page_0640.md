Use the available DISP mode value obtained by Monitor DISP (Screen Display) Mode Candidates(Movie)

```c
You must select “No Disp. Info. Exposure: On (0x0020)” or “No Disp. Info. Exposure: Timeout (0x0040)”
```

```c
If both “No Disp. Info. Exposure: On (0x0020)” and “No Disp. Info. Exposure: Timeout (0x0040)” are set,
“No Disp. Info. Exposure: On (0x0020)” will be preferred.
If neither “No Disp. Info. Exposure: On (0x0020)” nor “No Disp. Info. Exposure: Timeout (0x0040)” are
set, “No Disp. Info. Exposure: On (0x0020)” will be set.
Only when “No Disp. Info. (0x0010)” is selected can “No Disp. Info. Exposure: On (0x0020)” and “No
Disp. Info. Exposure: Timeout (0x0040)” be set.
```
