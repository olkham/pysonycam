<CameraSettingConfirmation>
“isMatch” attribute
If the “Param” in the “SDIGeneralSettingFile” matches the Camera settings, it is represented as
follows:
```c
<Item id="0x00000020">
    <Param id="0x2000000B" isMatch="True"/>
```
</Item>
If the settings do not match, it is represented as follows:
```c
<Item id="0x00000020">
    <Param id="0x2000000B" isMatch="False"/>
```
</Item>
