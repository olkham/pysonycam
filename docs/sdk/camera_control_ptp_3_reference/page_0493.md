Note
The device can get the effective range of live view with this property.
The device must set the coordinates within this range when setting the coordinates with
```c
SDIO_SetExtDevicePropValue (0x9205) or SDIO_ControlDevice (0x9207).
```
The x-coordinate is set in the upper two bytes and the y-coordinate is set in the lower two bytes
This value does not depend on the aspect ratio of the camera.
```c
The current value is not available (fixed 0x00000000).
```
