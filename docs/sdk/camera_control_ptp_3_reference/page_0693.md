SDIE_ObjectAdded
Summary
Notify that a shot file is ready to transfer.
Description
```c
Event Code: 0xC201
```
Parameter1: ObjectHandle
Parameter2: None
Parameter3: None
Note
The host can send GetObjectInfo with the notified ObjectHandle after receiving this event.
```c
The host can choose from checking the value of the Shooting File Info (0xD215) or waiting for this event
```
to send GetObjectInfo.

Summary
Notify that a shot file is deleted.
The device sends this event when the shot file is transferred.
Description
```c
Event Code: 0xC202
```
