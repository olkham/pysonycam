CameraSetting File
SDIO_ControlGeneralSettingFile and SDIO_GetControlGeneralSettingResultFile can be used to change
and confirm camera settings such as FTP settings.
SDIGeneralSettingFile and GeneralSettingResult are XML files.
SDIGeneralSettingFile
This file is generated by the User App and sent to the Camera.
It is used to transmit the settings requested by the User App to the Camera.
Example of SDIGeneralSettingFile:
<?xml version="1.0" encoding="UTF-8"?>
<CameraSetting>
<FileFormatVersion version="1"/>
<Group>
```c
    <Item id="0x01001000" label="FileTransfer.ServerSettings" preset="1">
      <Param id="0x01001F01" label="DisplayName"          value="News07"/>
      <Param id="0x01001F02" label="HostName"             value="FTPHostServer6"/>
      <Param id="0x01001203" label="Port"                 value="21"/>
      <Param id="0x01001F04" label="UserName"             value="cameraman01"/>
      <Param id="0x01001F05" label="Password"             value="wqeda4qef"/>
      <Param id="0x01001206" label="PassiveMode"          value="0"/>
      <Param id="0x01001F07" label="DestinationDirectory" value="/news07/cameraman01"/>
      <Param id="0x01001208" label="UsingSecureProtocol"  value="0"/>
```
</Item>
```c
    <Item id="0x01002000" label="FileTransfer.DefaultUploadServer">
       <Param id="0x01002F01" label="DefaultUploadServerSelect" value="News07"/>
```
</Item>
</Group>
<Group>
```c
     <Item id="0x01003000" label="FileTransfer.AutoUploadProxy">
       <Param id="0x01003201" label="AutoUploadProxy" value="1"/>
```
</Item>
</Group>
<Group>
```c
     <Item id="0x01004000" label="FileTransfer.AutoUpload">
       <Param id="0x01004201" label="AutoUpload" value="1"/>
```
</Item>
</Group>
</CameraSetting>
